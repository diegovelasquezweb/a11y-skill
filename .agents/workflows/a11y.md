# Accessibility Audit Workflow (Shared Standard)

1. **Target Identification**: Specify the base URL to audit.
2. **Execution**: Run the audit pipeline:
   ```bash
   node scripts/run-audit.mjs --base-url <URL>
   ```
3. **Capture**: Identify the `REPORT_PATH` from the command output.
4. **Reporting**:
   - Report the absolute path to `audit/report.html`.
   - Provide a high-level summary of Critical and High issues.
   - For remediation, proceed by reading `audit/remediation.md`.
